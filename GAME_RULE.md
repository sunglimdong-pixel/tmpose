# GAME_RULE.md - Fruit Catcher

## 1. 게임 개요 (Game Overview)
- **게임 제목**: Fruit Catcher (프루트 캐쳐)
- **장르**: 아케이드 / 포즈 인식 리듬 게임
- **설명**: 하늘에서 떨어지는 맛있는 과일들을 바구니로 받아내는 게임입니다. 폭탄을 피하고 가능한 많은 과일을 먹으세요!

## 2. 조작 방법 (Controls)
플레이어는 웹캠 앞에서 몸을 움직여 바구니(Player)를 조작합니다.
화면은 3개의 레인(Lane)으로 나뉩니다.

| 포즈 (Pose) | 행동 (Action) | 설명 |
|---|---|---|
| **LEFT** | 왼쪽 이동 | 바구니를 왼쪽 레인으로 이동시킵니다. |
| **CENTER** | 중앙 이동 | 바구니를 중앙 레인으로 이동시킵니다. |
| **RIGHT** | 오른쪽 이동 | 바구니를 오른쪽 레인으로 이동시킵니다. |

> **참고**: Teachable Machine에서 `LEFT`, `CENTER`, `RIGHT` 3가지 클래스를 학습시켜야 합니다.

## 3. 게임 규칙 (Rules)

### 3.1 아이템 (Items)
하늘에서 랜덤하게 아이템이 떨어집니다.

| 아이템 | 점수 | 효과 |
|---|---|---|
| 🍎 **사과 (Apple)** | +100점 | 기본 점수 획득 |
| 🍌 **바나나 (Banana)** | +200점 | 높은 점수 획득 |
| 🍍 **파인애플 (Pineapple)** | +300점 | 최고 점수 획득 |
| 💣 **폭탄 (Bomb)** | - | **생명(Life) 1 감소** (0이 되면 게임 오버) |

### 3.2 게임 플레이 주기 (Gameplay Loop)
1. 게임이 시작되면 바구니는 중앙(CENTER)에 위치합니다.
2. 과일과 폭탄이 랜덤한 레인에서 떨어지기 시작합니다.
3. 플레이어는 포즈를 취해 해당 레인으로 바구니를 이동시켜 과일을 받습니다.
4. 폭탄을 받으면 생명이 깎입니다.
5. 과일을 바닥에 떨어뜨려도 생명이 깎이지는 않지만, 점수를 얻지 못합니다. (옵션: 놓치면 콤보 끊김)

### 3.3 레벨 시스템 (Level System)
- **레벨 업**: 500점마다 레벨이 올라갑니다.
- **난이도 증가**: 레벨이 오를수록 아이템이 떨어지는 속도가 빨라집니다.

### 3.4 게임 오버 (Game Over)
- **생명 소진**: 폭탄을 3번 받으면 게임이 종료됩니다.
- **시간 제한**: (선택 사항) 60초 동안 최대한 높은 점수를 내는 모드로 설정할 수 있습니다. (기본은 무제한/생명제)

## 4. UI 요소 (User Interface)
- **Score**: 현재 점수 표시
- **Life**: 남은 생명 (하트 아이콘 ❤️❤️❤️)
- **Level**: 현재 레벨
- **Feedback**: 아이템 획득 시 "+100", 폭탄 피격 시 "OUCH!" 등의 시각적 효과 표시

## 5. Teachable Machine 모델 요구사항
- **Class 1**: `LEFT` (왼손을 들거나 왼쪽으로 기울이기)
- **Class 2**: `CENTER` (차렷 또는 정면)
- **Class 3**: `RIGHT` (오른손을 들거나 오른쪽으로 기울이기)
